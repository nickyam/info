如何在 Mac 上隐藏文件（夹）的技巧

本文最后更新于 2024年4月1日 下午

Mac为了防止用户意外更改或删除Mac运行所需的重要数据导致系统出现一些不可预知的问题，会默认隐藏一些文件和文件夹。如果我们个人有隐藏文件或文件夹的需要，该怎么办呢？

拖入到系统隐藏文件夹

  • 通过按快捷键 Shift + Command + . 显示系统隐藏的文件夹。
  • 将你想要隐藏的文件或文件夹拖入到Mac系统中原有的隐藏文件夹中，比如资源库。

新建隐藏文件

  • 打开终端（Terminal），在终端中输入 mkdir 文件位置/.文件名称，如mkdir desktop/.macbox，按回车键执行命令，即可创建一个隐藏文件夹（.macbox）。

  • 通过按下键盘快捷键 Shift + Command + .就可以将这个隐藏文件夹显示出来，进行进一步的操作。

  • 当然，你也可以在显示系统隐藏文件夹的时候复制某个系统隐藏文件夹，然后重新命名，也是一种新建的方式。

chflags命令修改文件隐藏属性

  • 在终端中输入 chflags hidden+空格，然后拖入要隐藏的文件，按回车键执行可隐藏该文件。
  • 按照上面相同的命令，将hidden更改为 nohidden，拖入要解除隐藏的文件，按回车键执行即可。

使用mv命令隐藏文件夹

mv命令在终端可以将文件夹从一个目的地传输到另一个目的地。 因此可调用“mv”命令将文件夹转移到句点文件夹。

启动终端并输入 mv filename .filename。

其中，filename参数替换为你需要隐藏的文件夹名即可。。
